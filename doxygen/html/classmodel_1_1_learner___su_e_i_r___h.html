Detailed Description

Class for SuEIR_H (Susceptible, Unreported, Exposed, Infected, Recovered, Hospitalized) model learner.

Hospitalization Prediction modified from SuEIR model: by removing exponential estimation dS = -beta S(E + I) / N dE = beta S(E + I) / N - sigma E dI = mu E - gamma I

dR = gamma I

dH = alpha I - rho H

dD = theta H

pred_S, pred_E, pred_I, pred_R, pred_confirm, pred_fatality, pred_hospital as S, E, I, R, I + R, D, H
